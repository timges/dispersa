DTCG token types fall into two categories: **primitive types** (single values) and **composite types** (structured objects combining primitives). [Color](https://www.designtokens.org/TR/2025.10/color/) tokens have their own spec module; all other types live in the [format module](https://www.designtokens.org/TR/2025.10/format/).

## Primitive Types

### color

A [color value](https://www.designtokens.org/TR/2025.10/color/#color-type) in a given color space. Component ranges depend on the color space (e.g. sRGB uses 0--1 per channel; HSL uses 0--360 for hue and 0--100 for saturation/lightness).

```json
{
  "$type": "color",
  "$value": {
    "colorSpace": "srgb",
    "components": [0, 0.4, 0.8],
    "alpha": 1
  }
}
```

Supported color spaces: `srgb`, `srgb-linear`, `display-p3`, `a98-rgb`, `prophoto-rgb`, `rec2020`, `hsl`, `hwb`, `lab`, `lch`, `oklab`, `oklch`, `xyz-d50`, `xyz-d65`. The `alpha` property is optional and defaults to `1`. An optional `hex` property can provide a 6-digit CSS hex fallback (e.g. `"#ff00ff"`). Individual components may also be the string `"none"` to indicate a missing or powerless component (e.g. hue in an achromatic color).

### dimension

A [numeric value with a unit](https://www.designtokens.org/TR/2025.10/format/#dimension). Supports `px` and `rem`.

```json
{
  "$type": "dimension",
  "$value": {
    "value": 16,
    "unit": "px"
  }
}
```

### fontFamily

A [font family](https://www.designtokens.org/TR/2025.10/format/#font-family) name or fallback stack.

```json
{
  "$type": "fontFamily",
  "$value": ["Inter", "sans-serif"]
}
```

### fontWeight

[Font weight](https://www.designtokens.org/TR/2025.10/format/#font-weight) as a number (1--1000) or named string (e.g. `thin`, `normal`, `bold`, `extra-bold`).

```json
{
  "$type": "fontWeight",
  "$value": "bold"
}
```

### duration

A time [duration](https://www.designtokens.org/TR/2025.10/format/#duration) in milliseconds or seconds.

```json
{
  "$type": "duration",
  "$value": {
    "value": 200,
    "unit": "ms"
  }
}
```

### cubicBezier

A [cubic Bezier curve](https://www.designtokens.org/TR/2025.10/format/#cubic-bezier) defined by four control points.

```json
{
  "$type": "cubicBezier",
  "$value": [0.25, 0.1, 0.25, 1]
}
```

### number

A plain [numeric value](https://www.designtokens.org/TR/2025.10/format/#number) without units.

```json
{
  "$type": "number",
  "$value": 1.5
}
```

## Composite Types

### shadow

A box [shadow](https://www.designtokens.org/TR/2025.10/format/#shadow). Use an array for multiple shadows. The optional `inset` property (default `false`) controls whether the shadow renders inside the element as an inner shadow.

```json
{
  "$type": "shadow",
  "$value": {
    "offsetX": { "value": 0, "unit": "px" },
    "offsetY": { "value": 4, "unit": "px" },
    "blur": { "value": 8, "unit": "px" },
    "spread": { "value": 0, "unit": "px" },
    "color": {
      "colorSpace": "srgb",
      "components": [0, 0, 0],
      "alpha": 0.25
    },
    "inset": true
  }
}
```

### typography

A combination of [font properties](https://www.designtokens.org/TR/2025.10/format/#typography).

```json
{
  "$type": "typography",
  "$value": {
    "fontFamily": ["Inter", "sans-serif"],
    "fontSize": { "value": 16, "unit": "px" },
    "fontWeight": 600,
    "lineHeight": 1.5,
    "letterSpacing": { "value": 0.16, "unit": "rem" }
  }
}
```

### border

[Border](https://www.designtokens.org/TR/2025.10/format/#border) color, width, and style.

```json
{
  "$type": "border",
  "$value": {
    "color": {
      "colorSpace": "srgb",
      "components": [0, 0.4, 0.8]
    },
    "width": { "value": 1, "unit": "px" },
    "style": "solid"
  }
}
```

### strokeStyle

[Stroke](https://www.designtokens.org/TR/2025.10/format/#stroke-style) configuration for vectors. Can be a string or an object with `dashArray` and `lineCap`.

```json
{
  "$type": "strokeStyle",
  "$value": {
    "dashArray": [
      { "value": 4, "unit": "px" },
      { "value": 2, "unit": "px" }
    ],
    "lineCap": "round"
  }
}
```

### transition

Animation [transition](https://www.designtokens.org/TR/2025.10/format/#transition) properties.

```json
{
  "$type": "transition",
  "$value": {
    "duration": { "value": 200, "unit": "ms" },
    "delay": { "value": 0, "unit": "ms" },
    "timingFunction": [0.25, 0.1, 0.25, 1]
  }
}
```

### gradient

A [gradient](https://www.designtokens.org/TR/2025.10/format/#gradient) defined by color stops.

```json
{
  "$type": "gradient",
  "$value": [
    {
      "color": { "colorSpace": "srgb", "components": [0, 0.4, 0.8] },
      "position": 0
    },
    {
      "color": { "colorSpace": "srgb", "components": [0, 0.8, 0.4] },
      "position": 1
    }
  ]
}
```
